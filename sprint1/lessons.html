Html lesson 01 - Html page structure

Lets start from basics and understand what all of these tags and their attributes mean:

  • <!DOCTYPE html> declaration defines that this document is an HTML5 document.
  • The "html" tag represents the root of an HTML document. The "html" tag is the container for all other HTML elements and "lang" attribute declares the language.
  • The "head" element contains meta information about the HTML page.
  • <meta charset="UTF-8"> tells the page what kind of symbols will be used.
  • <meta http-equiv="X-UA-Compatible" content="IE=edge"> allows web authors to choose what version of Internet Explorer the page should be rendered as.
  • <meta name="viewport" content="width=device-width, initial-scale=1.0"> gives the browser instructions on how to control the page's dimensions and scaling.
  • The "title" element specifies a title for the HTML page which is shwon in the browser's title bar or in the page's tab.
  • The "body" element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc. 
            
            <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta http-equiv="X-UA-Compatible" content="IE=edge">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Document<title>
    </head>
    <body>
        
    </body>
    </html>
